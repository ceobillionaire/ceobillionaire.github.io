Routage recommandé

  • Pour les engagements QUEBEC.AI : inclure l’organisation, le rôle, l’objectif, l’échéancier et le contexte stratégique.
  • Pour les médias : inclure le média, l’échéance, le sujet, le format et la fenêtre de citation ou d’entrevue demandée.
  • Pour les questions techniques : inclure le dépôt, le protocole ou la page source concernés.
  • Pour les questions d’archive : inclure l’URL ou le nom du fichier concerné.
