Confidentialité

Site minimal. Données minimales.

VincentBoucher.com est un site public statique. Il présente de l’information publique, des matériaux médias, des routes institutionnelles et des options de correspondance sans exiger de compte utilisateur, formulaire, base de données ou script de suivi.

Avis de confidentialité.

Ce site n’exige pas de connexion, ne traite pas de paiements, n’exploite pas de base de données utilisateur et n’utilise pas de formulaire de contact. Si vous contactez QUEBEC.AI ou Vincent Boucher par courriel, votre message est traité selon le contexte de la correspondance et le canal choisi.
